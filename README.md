# What is Tic-Tac-Toe?
Tic-tac-toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner - Wikipedia
# How do I play it?
1) Download or Clone the repository
2) Unzip the files
3) Open Command Prompt
4) Type "cd [your-directory-name]"
5) Type "pip install -r requirements.txt"
6) Type "python main.py"
